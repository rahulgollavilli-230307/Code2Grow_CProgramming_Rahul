# Student Result System

## Author
**G. Rahul (AP25110090193)**

## Description
This is a simple C program to calculate and display student results and class summary.

### Features
- Input marks for 3 subjects per student
- Calculate total, average, and grade
- Count passed and failed students
- Display highest, lowest, and class average
- Menu for viewing summary or exiting

## How to Run
1. Open terminal in this directory.
2. Compile the program:
   ```bash
   gcc main.c -o build/student_results
   ```
3. Run the program:
   ```bash
   ./build/student_results
   ```
