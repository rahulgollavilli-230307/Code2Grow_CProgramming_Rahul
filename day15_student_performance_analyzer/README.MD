# Student Performance Analyzer

## Description
This C program analyzes the performance of students based on their marks.

## Features
- Accepts marks for up to 10 students.
- Calculates total, average, highest, and lowest marks.
- Counts number of students who passed and failed.
- Displays grade distribution (A+, A, B, C, D, Fail).

## Compilation
```bash
gcc main.c -o analyzer
```

## Execution
```bash
./analyzer
```
