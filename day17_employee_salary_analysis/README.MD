# Employee Salary Analysis System

## Description
A C program that allows HR to analyze employee salaries using a menu-driven system.  
Each run is tagged with student identification for submission verification.

## Features
- Input student name, registration number, and section.
- Store and analyze up to 10 employee salaries.
- Display total, average, highest, and lowest salaries.
- Count employees above and below the average.
- Sort salaries in ascending order.

## How to Build and Run
```bash
gcc main.c -o salary_analysis
./salary_analysis
```
