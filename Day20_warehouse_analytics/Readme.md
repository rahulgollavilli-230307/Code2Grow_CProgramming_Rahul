# Warehouse Management System (C Program)

**Student:** G. Rahul

## Description
This project is a simple Warehouse Management System written in C.  
It allows the user to:
- Add products
- Search products
- Update product quantity
- Generate rack reports
- View stock value reports
- Display all products

## Files Included
- **main.c** — Full C source code
- **README.md** — Project description and instructions
- **Makefile** — To compile the program easily

## How to Compile
Run the command:

```
make
```

## How to Run
```
./program
```
