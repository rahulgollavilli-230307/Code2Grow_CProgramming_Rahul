
# Smart City Traffic Control System
**Author:** G. Rahul  
**Reg No:** AP25110090193  

This program records vehicle details such as:
- Vehicle ID  
- Speed  
- Lane  

It then performs:
- Speed violation analysis  
- Lane violation analysis  
- Traffic density analysis  
- Detailed violation report  

## How to Build
```
gcc main.c -o traffic
```

## How to Run
```
./traffic
```
