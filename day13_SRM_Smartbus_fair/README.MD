# Passenger Fare System

## Description
This C program calculates passenger fares based on distance, age discounts, and peak hour surcharges.

## Features
- Child and senior citizen discounts
- Peak hour surcharge (8–10 AM and 5–8 PM)
- Tracks total revenue, highest, and lowest fares

## Compilation
```bash
gcc main.c -o build/fare_system
```

## Execution
```bash
./build/fare_system
```
