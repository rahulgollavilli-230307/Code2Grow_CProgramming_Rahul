# Cafe Order Management System

## Description
A simple C program that simulates a cafe order management system.  
It allows users to:
- View and order from a menu of 5 items.
- Handle up to 10 customers.
- Calculate each customer's bill.
- Display total revenue, total items sold, and most/least ordered items.

## How to Compile and Run
1. Open terminal or command prompt.
2. Navigate to this project folder.
3. Compile using:
   ```bash
   gcc main.c -o cafe
   ```
4. Run the program:
   ```bash
   ./cafe
   ```
