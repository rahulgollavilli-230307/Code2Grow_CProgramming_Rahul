# Banking System - C Program

### Author
**G. Rahul (AP25110090193)**

### Description
A simple C program that simulates a basic banking system. It allows a user to:
- Check balance
- Deposit money
- Withdraw money
- Exit the system

The program limits the number of transactions to 5 (excluding exit).

### How to Run
1. Compile the code using GCC:
   ```bash
   gcc main.c -o build/banking_app
   ```
2. Run the program:
   ```bash
   ./build/banking_app
   ```
